默认启用的随机化配置**（位于 `EventCfg`类中）包含以下 9 项，均在训练的不同阶段自动触发：


| 事件名称                                | 触发模式              | 随机化内容                     | 参数详情                                                                                       |
| --------------------------------------- | --------------------- | ------------------------------ | ---------------------------------------------------------------------------------------------- |
| `randomize_rigid_body_material`         | startup（环境初始化） | 刚体物理材质（摩擦、恢复系数） | 静态摩擦：(0.3, 1.0)<br/>动态摩擦：(0.3, 0.8)<br/>恢复系数：(0.0, 0.5)<br/>分桶数：64          |
| `randomize_rigid_body_mass_base`        | startup               | 机器人基座质量                 | 质量偏移范围：(-1.0, 3.0) kg<br/>操作：`add`（叠加）<br/>重新计算惯性：是                      |
| `randomize_rigid_body_mass_others`      | startup               | 其余身体部件质量               | 缩放范围：(0.7, 1.3) 倍<br/>操作：`scale`（比例缩放）<br/>重新计算惯性：是                     |
| `randomize_com_positions`               | startup               | 质心位置                       | XYZ 偏移范围：±0.05 m                                                                         |
| `randomize_apply_external_force_torque` | reset（每回合重置）   | 外部力与扭矩                   | 力范围：±15.0 N<br/>扭矩范围：±15.0 N·m<br/>作用于基座                                      |
| `randomize_reset_joints`                | reset                 | 关节初始状态                   | 位置缩放：(1.0, 1.0)（无随机）<br/>速度范围：(0.0, 0.0)（无随机）                              |
| `randomize_actuator_gains`              | reset                 | 执行器刚度与阻尼               | 刚度缩放：(0.5, 2.0) 倍<br/>阻尼缩放：(0.5, 2.0) 倍<br/>均匀分布                               |
| `randomize_reset_base`                  | reset                 | 机器人根状态（位姿与速度）     | 位置：XY ±0.5 m，偏航角 ±π<br/>线速度：XYZ ±0.5 m/s<br/>角速度：滚转/俯仰/偏航 ±0.5 rad/s |
| `randomize_push_robot`                  | interval（周期性）    | 随机水平推力                   | 触发间隔：10.0–15.0 秒<br/>速度扰动：XY ±0.5 m/s                                             |
